Arquivo de Ajuda - Petshop Animalia

Este arquivo tem como objetivo explicar o funcionamento do site do Petshop Animalia e orientar o usuário sobre como navegar entre as páginas.

Página inicial| Produtos | Serviços Ajuda

1. Estrutura do Site

O site é composto pelas seguintes páginas:

  • index.html – Página inicial do site.
  • produtos.html – Lista de produtos disponíveis.
  • servicos.html – Lista de serviços oferecidos.
  • ajuda.html – Página atual, contendo orientações de uso.

2. Navegação

Em todas as páginas existe um menu no topo e rodapé contendo links para as outras seções do site:

  • Início – retorna para a página inicial.
  • Produtos – abre a página de produtos.
  • Serviços – abre a página de serviços.
  • Ajuda – abre esta página de ajuda.

Para navegar entre as páginas, basta clicar nos links presentes no menu.

3. Página de Produtos

A página de produtos apresenta uma lista com os itens vendidos, contendo nome, descrição e preço de cada produto.

4. Página de Serviços

A página de serviços apresenta os serviços oferecidos pelo petshop, também com nome, descrição e valor.

5. Como voltar à página inicial

Para retornar à página inicial, clique no link Início que aparece no menu superior.
